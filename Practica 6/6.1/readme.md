# Descripción 

En esta práctica se trabajó con un display LCD como medio de visualización de texto y caracteres personalizados utilizando el microcontrolador PIC16F887. El objetivo fue comprender la comunicación y configuración básica de un LCD, así como la manipulación de pantallas y la interacción con entradas digitales para cambiar entre diferentes visualizaciones. Las actividades permitieron reforzar el manejo de periféricos de salida y la creación de interfaces simples con el usuario.

_Hardware utilizado_
- Microcontrolador PIC16F887
- Programador PICkit
- Display LCD 16x2
- Botón pulsador
- Protoboard
- Resistencias
- Cables de conexión

_Software utilizado_ 
- MPLAB X IDE
- Compilador XC8
- Proteus Design Suite

# Ejercicios realizados

1. Visualización de texto dinámico: Se configuró un display LCD de 16x2 para mostrar texto en ambas líneas. En la primera línea se desplegó el mensaje "HELLO WORLD", mientras que en la segunda línea se implementó una secuencia en la que las letras A a P aparecen progresivamente una por una. Este ejercicio permitió practicar la escritura de caracteres y el control dinámico del contenido en pantalla.

2. Cambio de pantallas mediante botón: Se implementó un sistema de cambio de pantallas utilizando un botón como entrada digital. Cada vez que el botón es presionado, el LCD alterna entre dos pantallas diferentes: la primera muestra caracteres personalizados representando los fantasmitas de Pac-Man, mientras que la segunda despliega el mensaje "WAKA WAKA". Este ejercicio permitió integrar la lectura de entradas con la actualización dinámica de la interfaz en el LCD.
