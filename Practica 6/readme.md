# Práctica 6 - Manejo de Display LCD

En esta práctica se trabajó con un display LCD como medio de visualización de texto y caracteres personalizados utilizando el microcontrolador PIC16F887. El objetivo fue comprender la configuración y comunicación básica con el LCD, así como la creación de caracteres especiales y su uso en animaciones simples mediante interacción con entradas digitales. Las actividades permitieron reforzar el manejo de periféricos de salida y la personalización de interfaces visuales.

_Hardware utilizado_
- Microcontrolador PIC16F887
- Programador PICkit
- Display LCD 16x2
- Botón pulsador
- Protoboard
- Resistencias
- Cables de conexión

_Software utilizado_
- MPLAB X IDE
- Compilador XC8
- Proteus Design Suite

# Ejercicios realizados

1. Visualización de "Hello World!": Se configuró el display LCD para mostrar el mensaje **"Hello World!"** como prueba básica de comunicación entre el microcontrolador y el periférico. Este ejercicio permitió familiarizarse con el envío de caracteres y el posicionamiento de texto en pantalla.

2. Carácter personalizado y animación: Se diseñó un carácter especial personalizado dentro de la memoria CGRAM del display LCD. Posteriormente, al presionar un botón, se activaba una animación utilizando dicho carácter, desplazándolo o modificándolo en pantalla. Este ejercicio permitió explorar la personalización gráfica básica y la interacción mediante entradas digitales.
