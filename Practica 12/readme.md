# Práctica 12 - Teclado Matricial 

En esta práctica se trabajó con un teclado matricial como dispositivo de entrada para el microcontrolador PIC16F887. El objetivo fue comprender el proceso de escaneo de filas y columnas para detectar teclas presionadas, así como integrar la información obtenida en un display LCD. Las actividades permitieron reforzar conceptos de entradas digitales y diseño de interfaces de usuario.

_Hardware utilizado_
- Microcontrolador PIC16F887
- Programador PICkit
- Teclado matricial
- Display LCD 16x2
- Protoboard
- Resistencias
- Cables de conexión

_Software utilizado_
- MPLAB X IDE
- Compilador XC8
- Proteus Design Suite

# Ejercicios realizados

1. Visualización de teclas en LCD: Se implementó la lectura de un teclado matricial y la visualización en tiempo real de los caracteres presionados en un display LCD. Esto permitió validar la correcta detección de entradas y su representación en pantalla.

2. Calculadora básica: Se desarrolló una calculadora básica capaz de realizar operaciones de suma, resta, multiplicación y división entre dos números ingresados mediante el teclado matricial. Los resultados fueron mostrados en el display LCD, integrando entrada y salida en una sola aplicación.
