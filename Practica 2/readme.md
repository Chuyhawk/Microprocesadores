# Práctica 2 - Control de Matriz de LEDs 8x8

En esta práctica se trabajó con una matriz de LEDs de 8x8 controlada mediante el microcontrolador PIC16F887. El objetivo fue comprender el funcionamiento y direccionamiento de este tipo de dispositivos, implementando patrones visuales y caracteres alfanuméricos mediante el control de filas y columnas. Las actividades permitieron reforzar el manejo de salidas digitales y la generación de secuencias visuales utilizando programación embebida.

_Hardware utilizado_
- Microcontrolador PIC16F887
- Programador PICkit
- Matriz de LEDs 8x8
- Protoboard
- Resistencias limitadoras de corriente
- Cables de conexión

_Software utilizado_ 
- MPLAB X IDE
- Compilador XC8
- Proteus Design Suite

# Ejercicios realizados 

1. Visualización de una "X": Se implementó un patrón fijo sobre la matriz de LEDs para representar la letra "X". Mediante la activación selectiva de filas y columnas se logró formar la figura deseada, permitiendo comprender la forma en que se generan imágenes dentro de una matriz de LEDs.

2. Despliegue secuencial de caracteres: Se desarrolló una secuencia cíclica de caracteres en la matriz de LEDs, mostrando las letras A, N, C y H de manera consecutiva. Para cada letra se definió un patrón específico de iluminación, demostrando la capacidad del sistema para representar distintos caracteres mediante el control programado de la matriz.
