# Práctica 13 - Control de Motores DC

En esta práctica se trabajó con motores de corriente directa (DC), enfocándose en el control de velocidad y dirección mediante diferentes configuraciones de potencia. El objetivo fue comprender la interacción entre el microcontrolador y circuitos de manejo de carga, así como la aplicación de PWM para regular el comportamiento del motor.

_Hardware utilizado_
- Microcontrolador PIC16F887
- Programador PICkit
- Motor DC
- Transistores BJT
- Puente H
- Fuente de alimentación
- Protoboard
- Resistencias
- Cables de conexión

_Software utilizado_
- MPLAB X IDE
- Compilador XC8
- Proteus Design Suite

# Ejercicios realizados

1. Control de velocidad con driver basado en BJT: Se implementó un circuito controlador utilizando transistores BJT para regular la velocidad de un motor DC. Mediante señales PWM generadas por el microcontrolador, fue posible modificar la velocidad de giro del motor.

2. Control de giro y velocidad con puente H: Se desarrolló un sistema utilizando un puente H para controlar tanto la velocidad como la dirección de giro del motor DC. Esto permitió invertir el sentido de rotación y variar la velocidad según las señales enviadas desde el microcontrolador.
