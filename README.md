# Microprocesadores
Practicas de la materia Microprocesadores VR'26

Las practicas en este repositorio se hicieron utilizando el microcontrolador *PIC16F887*.

Para cada una de las practicas se realizaron en 3 partes:
* Programa realizado en MPLab IDE (extensión .hex)
* Simulación realizada en Proteus (extension .pdsprj)
* Una fotografía del circuito como forma de evidencia

Las prácticas están organizadas por número en sus respectivos apartados. Cada práctica se divide en ejercicios que incluyen su propia explicación. Además, dentro de la subcarpeta de cada práctica encontrará los 3 archivos mencionados.
