# Microprocesadores
Prácticas de la materia Microprocesadores VR'26

Las prácticas en este repositorio se hicieron utilizando el microcontrolador *PIC16F887*.

Para cada una de las practicas se realizó:
* Un programa en MPLab IDE (extensión .hex)
* Una simulación en Proteus (extension .pdsprj)
* Una fotografía del circuito como forma de evidencia

Las prácticas están organizadas por número en sus respectivos apartados. Cada práctica se divide en ejercicios que incluyen su propia explicación. Además, dentro de la subcarpeta de cada práctica se encuentran los 3 archivos mencionados.
