# Práctica 11 - PWM por Hardware y Software

En esta práctica se trabajó con la generación de señales PWM (Pulse Width Modulation) utilizando el microcontrolador PIC16F887, tanto mediante módulos de hardware dedicados como por implementación en software. El objetivo fue comprender cómo variar la potencia entregada a una carga mediante la modificación del ciclo de trabajo, aplicándolo al control de intensidad de LEDs. Esta práctica permitió reforzar conceptos de temporización, modulación y control analógico simulado.

_Hardware utilizado_
- Microcontrolador PIC16F887
- Programador PICkit
- LEDs
- 2 potenciómetros
- Resistencias
- Protoboard
- Cables de conexión

_Software utilizado_
- MPLAB X IDE
- Compilador XC8
- Proteus Design Suite

# Ejercicios realizados

1. Control de intensidad de un LED con PWM Hardware: Se implementó un sistema de control de brillo de un LED utilizando el módulo PWM por hardware del PIC16F887. Un potenciómetro fue utilizado para variar el ciclo de trabajo de la señal PWM, modificando así la intensidad luminosa del LED de manera proporcional.

2. Control de intensidad de dos LEDs con dos potenciómetros: Se amplió la práctica implementando el control independiente de dos LEDs mediante dos potenciómetros distintos. Cada entrada analógica modificaba la intensidad de un LED, permitiendo trabajar con múltiples canales y reforzar el uso simultáneo de PWM y ADC.
